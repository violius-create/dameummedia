## 선택된 디자인 방향

**스위스 국제 타이포그래피 스타일**을 선택한다. 담음미디어의 분석 데이터와 전략적 제안을 명확하고 전문적으로 전달하기 위해서는 정보의 계층 구조가 명확하고 가독성이 뛰어난 디자인이 필요하다. 이 스타일은 데이터 중심의 콘텐츠를 효과적으로 표현하며, 신뢰감과 전문성을 동시에 전달할 수 있다.

### 구체적 적용 방안

**색상 팔레트**:
- Primary: #1a1a1a (Deep Charcoal)
- Background: #f5f5f5 (Light Gray)
- Accent: #0066cc (Professional Blue)
- Text: #2c2c2c (Dark Gray)
- Muted: #6b6b6b (Medium Gray)

**타이포그래피**:
- Display: Inter Bold 700
- Heading: Inter SemiBold 600
- Body: Inter Regular 400
- Caption: Inter Regular 400 (smaller size)

**레이아웃 구조**:
- 고정 사이드바 네비게이션 (240px)
- 메인 콘텐츠 영역은 최대 1200px
- 8px 그리드 시스템 적용
- 섹션 간 구분은 1px 실선 사용

**주요 UI 요소**:
- 카드: 흰색 배경, 미묘한 그림자, 4px 라운딩
- 버튼: 솔리드 배경, 호버 시 약간 어두워짐
- 데이터 시각화: 단순한 막대 그래프와 라인 차트
- 아이콘: Lucide React의 미니멀한 아이콘 세트
